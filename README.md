# 押しモグラ

叩いてもモグラは死なない。隣の穴へ飛ぶだけ。モグラ同士をぶつけた時だけ消える。

盤外へは飛ばない。除去する手段は衝突ただ一つ。

遊ぶ: https://mirai-x64.github.io/oshi-mogura/

## 規則

- 穴を叩くと、中のモグラは**叩いた点の反対側**へ隣の穴 1 マスぶん飛ぶ
- 飛んだ先が空なら居座る
- 飛んだ先にモグラがいれば**2匹とも弾けて消える**
- **盤外へは飛ばない。**壁に当たって元の穴に落ちる(向きが変わるだけの空振り)
- 全16穴が埋まったら終了

除去する手段は衝突ただ一つ。端へ押し出して逃がす抜け道は無い。

## 確かめたこと

`node solver.mjs` — spawn を止めた静的盤面を全探索する。
状態は 16bit なので 65536 通りを全て潰せる。

    モグラ2匹以上のあらゆる盤面で消せない盤面: 0
    最悪ケースでも 6 手で1組消える

盤外が空振りなので総数は spawn でしか増えず衝突でしか減らないが、
それでも**詰みは存在しない**。「狙って消せる」は運ではなく全数で確かめてある。

`node verify.mjs` — 実ブラウザ(CDP)で実際にクリックし、
200 手の盤面遷移が solver の規則と一致するかを1手ずつ照合する。
移動・衝突・壁での空振りの3経路すべてを踏み、放置時に埋まって終わることと、
2手の狙い撃ち(空の穴へ寄せる→隣から押し込む)の再現も併せて見ている。
